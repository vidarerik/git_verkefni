#2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?

	*	aftritar annað repository
*	git log		
	*	Shows the commit logs
*	git status	
	*	Sýna hvað er búið að breyta eða hvað á eftir að commita
*	git diff		
	*	Sýnir það sem var mergaeað
